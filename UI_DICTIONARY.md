# UI Block Dictionary

This document maps unique UI identifiers to their corresponding interface locations.

| ID | Name | Description |
| :--- | :--- | :--- |
| **UI-001** | Top Navigation Header | The main header bar at the top of the application. |
| **UI-002** | App Logo Area | Contains the app branding and logo. |
| **UI-003** | Header Navigation Menu | The central navigation links (Dashboard, Markets, Settings). |
| **UI-004** | Header Search Bar | The search input field in the header. |
| **UI-005** | Header User Actions | Notification bell and user profile avatar. |
| **UI-006** | Sidebar Container | The left sidebar containing the watchlist. |
| **UI-007** | Watchlist Header | The header of the watchlist section ("Watchlist", "Live Market"). |
| **UI-008** | Watchlist List Container | The scrollable area containing the list of assets. |
| **UI-009** | Asset Item | Individual asset row in the watchlist. |
| **UI-010** | Watchlist Add Button | The footer area of the sidebar with the "Add Asset" button. |
| **UI-011** | Dashboard Main Area | The main content area on the right. |
| **UI-012** | Asset Header | The top banner showing the selected asset's symbol and name. |
| **UI-013** | Asset Info Card | The detailed information card (Symbol, Name, Category). |
| **UI-014** | Static Mode Card | The placeholder card explaining that live data is disabled. |
